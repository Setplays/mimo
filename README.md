# mimo

MIMO D'CACAU

Esse é um projeto do qual tenho muito orgulho. É um website real usado para vendas de doces caseiros, sejam de páscoas ou apenas chocolates comuns.

Para que o site funcione você deve apenas jogar tudo em uma pasta, e abrir o cmd dessa pasta. Depois disso execute:

php -S localhost:8000

Após isso você já poderá acessar o site no link localhost:8000

Para acessar a página de admin você deve acessar: localhost:8000/admin.php
